How to Install

Server Configuration:
  • PHP v5.4 / v5.5 / v5.6 / v7.0 & MySQL v5.x
  • Make sure curl, mbstring, allow_open_url, set_time_limit, mod_rewrite, mysqli support are enabled in your server.
  • Make sure open_basedir has no value set and safe_mode is on.

[Note: If you get "404 page not found" error then first make sure .htaccess file was uploaded properly in the app root and make AllowOverride in your virual host.]

Installation Manual:
  1. Download .zip package
  2. Upload it to your server
  3. Extract the package

  4. Please make sure that the following folders (and child files) or files have write permission (777 or 755)

    • Folder : application/config
    • Folder : application/core
    • Folder : upload
    • Folder : download
    • File : application/install.txt

  5. Run the installer url via browser (http://yourdomain.com/.../ytubelytics/) and you will find a interface to provide the settings for YTubeLytics:
    • Hostname : database host name / IP
    • Database Name : create a mysql database for YTubeLytics on your host and write that name here
    • Database username  : username of the created database
    • Database password : password of the created database
    • YTubeLytics Admin Panel Login Email: this will be used to login as admin
    • YTubeLytics Admin Panel Login Password : password to log in as admin
    • Company Name and Company Address
    • Company Phone /  Mobile : with country code
  6. Click Install YTubeLytics Now Button
  7. You are done. Log in with your admin username and password and start using YTubeLytics.

Keyword Scraper

This feature can grab any youtube video's keywords/tags. Simply put the youtube video id and click "Start Scraping", you will get the keywords. You can download the result as csv.

Cron Job Settings [Only Admin]

Video rank tracking and scheduled video upload needs cron jobs to complete time to time tasks. You have to set 2 cron jobs in your server. You will find the cron job commands in your admin panel > cron job settings menu. Cron job commands needs a API key key. Generate your api key then copy the cron job commands and set to your server.


How to Update

Update - v1.2 to v1.3

1. Download v1.3

2. Copy/Replace your following files and folders taking from v1.3

  • application/controllers/home.php
  • application/librarirs/google_youtube_login.php
  • application/librarirs/Google_youtube/Service/YouTubeAnalytics.php

3. Open application/config/my_config.php and update 'product_version' to 1.3

4. You are done.

Update - v1.1 to v1.2

1. Download v1.2

2. Copy/Replace your following files and folders taking from v1.2

  • application/controllers/home.php
  • application/controllers/youtube_analytics.php
  • documentation/*

3. Open application/config/my_config.php and update 'product_version' to v1.2

4. You are done.

Update - v1.0 to v1.1

1. Download v1.1

2. Copy/Replace your following files and folders taking from v1.1

  • application/controllers/home.php
  • application/controllers/youtube_analytics.php
  • application/views/page/*

  • documentation/*

3. Open application/config/my_config.php and update 'product_version' to v1.1
